# 📥 Uploads — Drop Your Files Here

> This folder is your **inbox**. You (the owner) drop image/video files here, and a developer moves them into the website. You do **not** need to know any code. Just put the right file with the right name in the right subfolder, then commit/upload to the repo.

## How it works (3 steps)

1. **Find what's needed** in the checklist below (also see `images/NEEDED.md` and `video/NEEDED.md`).
2. **Save your file with the exact name shown**, and place it in the matching subfolder:
   - Pictures → `uploads/images/`
   - Banner video → `uploads/video/`
3. **Upload it to the repo** (commit/push, or use GitHub's "Add file → Upload files" button on the web).

A developer then optimizes and moves it into the live site and ticks it off. That's it. 🙌

## What to give us (file format & size)

- **Photos:** `.jpg` (or `.png` if it needs transparency). Bigger is fine — we shrink them. Aim for at least the "min width" listed.
- **Banner video:** `.mp4`, landscape (16:9), short loop (8–15 sec), no important sound (it plays muted), ideally under ~15 MB.
- **Logo:** `.svg` if you have it, otherwise a high-res `.png`.
